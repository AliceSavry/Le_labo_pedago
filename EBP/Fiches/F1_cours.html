Fiche 1 - Création de dossier

Objectifs

Cette fiche vous guidera dans la création d'un nouveau dossier sur EBP Gestion Commerciale.

Étapes de création

  1. Lancer EBP Gestion Commerciale
  2. Créer un nouveau dossier
    • Cliquer sur "Créer un nouveau dossier"
    • Choisir le type de dossier
    • Renseigner les informations de base
  3. Paramétrer les informations société
    • Coordonnées
    • Identification
    • Options de base

Points d'attention

  • Bien vérifier les informations saisies
  • Sauvegarder régulièrement
  • Noter l'emplacement du dossier créé
